
⚠️  ВАЖНО: КОМПИЛЯЦИЯ НА WINDOWS
═════════════════════════════════════════════════════════════════════

❌ ПРОБЛЕМА:
Buildozer на Windows имеет ограничения и не поддерживает Android target "из коробки".
Это требует полной установки Android SDK, NDK и других инструментов, что сложно на Windows.

✅ РЕШЕНИЕ: Используйте один из этих вариантов:

═══════════════════════════════════════════════════════════════════════

ВАРИАНТ 1: ИСПОЛЬЗОВАТЬ WSL2 (Windows Subsystem for Linux)
═══════════════════════════════════════════════════════════════════════

1. Включите WSL2:
   PowerShell (Администратор):
   wsl --install
   
2. Перезагрузитесь

3. Откройте Linux терминал (Ubuntu):
   Перейдите в папку проекта и выполните:
   
   cd /mnt/c/Users/Артём/Desktop/папка\ Артёма/PYTON/браузер/браузер\ релиз/
   
4. Установите зависимости (в Ubuntu):
   sudo apt update
   sudo apt install -y python3-pip python3-dev python3-venv openjdk-11-jdk
   pip3 install --upgrade buildozer cython kivy requests
   
5. Компилируйте:
   buildozer android debug


═══════════════════════════════════════════════════════════════════════

ВАРИАНТ 2: ИСПОЛЬЗОВАТЬ DOCKER (самый простой)
═══════════════════════════════════════════════════════════════════════

1. Установите Docker Desktop для Windows:
   https://www.docker.com/products/docker-desktop

2. В папке проекта создайте Dockerfile:
   (смотрите файл Dockerfile в папке)

3. Откройте PowerShell и выполните:
   docker build -t nexus-apk .
   docker run -v C:\Users\Артём\Desktop\папка\ Артёма\PYTON\браузер\браузер\ релиз:/workspace nexus-apk

4. APK будет в папке bin/


═══════════════════════════════════════════════════════════════════════

ВАРИАНТ 3: ИСПОЛЬЗОВАТЬ ОНЛАЙН КОМПИЛЯТОР
═══════════════════════════════════════════════════════════════════════

Используйте облачный сервис:
- https://buildozer.io (если есть)
- GitHub Actions (бесплатно через .github/workflows)
- Kivy Cloud Build
